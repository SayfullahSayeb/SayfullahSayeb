যদি ভুলে কোন ব্রাঞ্চে পুশ হয়ে যায় তাহলে যেভাবে আগের ভার্সনে ব্যাক করবেন

January 12, 2025 · 1 min · Sayfullah Sayeb

প্রথমে, ভুলে পুশ করার আগের কমিট টা বের করতে হবে।

git log

ব্রাঞ্চ টি রিসেট করতে হবে সেই কমিটের সাথে 

git checkout main

git reset –hard

এর যায়গায় আপনি যে কমিট টি রিসেট করতে চাচ্ছেন তার হ্যাশটি দিতে হবে। 

এখন আপনি ফাইল মোডিফাই করতে পারেন  অথবা ফোরসফুলি পুশ করতে পারেন 

git push origin main –force
